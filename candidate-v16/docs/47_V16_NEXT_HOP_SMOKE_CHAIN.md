# v16 Next-Hop Smoke Chain

1. `bridge-smoke-run` confirms local bridge can operate.
2. `chrome-smoke-handoff` gives a compact Chrome Dev profile test.
3. `github-test-upload-handoff` gives a single test-repo upload route.
4. `auto-continue-live-handoff` wires the adapter without forcing it into the plugin.
5. `evidence-pack` stores compact JSON evidence from Codex or other agents.

The goal is not to slow Codex down. The goal is to let Codex query one API and immediately know the next executable step and required evidence.
