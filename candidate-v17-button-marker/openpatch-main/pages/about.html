About OpenPatch

Upload AI-generated code packages from ChatGPT to GitHub workflows.

Settings

What OpenPatch Does

OpenPatch connects ChatGPT-generated code files with GitHub Actions. It adds an Upload to GitHub button beside supported ChatGPT attachments, then sends `changed-files.zip`, `.diff`, or `.patch` files to the configured repository.

The browser extension only handles the upload. The target repository workflow applies the package, commits the real code changes, and removes temporary OpenPatch files.

Recommended Flow

  1. Copy the ZIP prompt from the floating toolbar on ChatGPT.
  2. Ask ChatGPT to generate `changed-files.zip`.
  3. Click Upload to GitHub beside the generated attachment.
  4. Let GitHub Actions apply and commit the code changes.

Supported Packages

  • changed-files.zip: recommended, contains complete changed files.
  • openpatch.diff: supported when generated by Git commands.
  • .patch: supported for compatibility.

OpenPatch 是什么

OpenPatch 用来把 ChatGPT 生成的代码修改包发送到 GitHub，再由目标仓库里的 GitHub Actions 自动应用修改。

插件只负责 upload，不在浏览器里解析或合并代码。真正的代码应用、提交和临时文件清理 都由仓库 workflow 完成。

推荐使用方式

  1. 在 ChatGPT 页面右上角 OpenPatch 浮窗复制 ZIP 提示词。
  2. 让 ChatGPT 生成 `changed-files.zip`。
  3. 点击附件旁边的 Upload to GitHub。
  4. 等待 GitHub Actions 自动应用并提交代码修改。

支持的文件

  • changed-files.zip：推荐方式，包含修改后的完整文件。
  • openpatch.diff：支持，但必须通过 Git 命令生成。
  • .patch：兼容旧 patch 文件。
